# Unit Converter Tool

## 📌 Project Overview

The Unit Converter Tool is a simple command-line application built using Python. It allows users to convert values between different units through an interactive menu-driven interface. This project demonstrates the use of Python fundamentals such as loops, conditional statements, user input, and arithmetic operations.

## ✨ Features

* 🌡️ Temperature Conversion

  * Celsius to Fahrenheit
  * Fahrenheit to Celsius
* 📏 Length Conversion

  * Meter to Kilometer
  * Kilometer to Meter
* ⚖️ Weight Conversion

  * Kilogram to Gram
  * Gram to Kilogram
* 🔄 Menu-driven interface
* 🚪 Exit option to close the application

## 🛠️ Technologies Used
